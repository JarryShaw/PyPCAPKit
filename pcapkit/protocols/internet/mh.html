MH - Mobility Header¶

pcapkit.protocols.internet.mh contains MH only, which implements extractor for Mobility Header (MH) [*], whose structure is described as below:

Octets

Bits

Name

Description

0

0

mh.next

Next Header

1

8

mh.length

Header Length

2

16

mh.type

Mobility Header Type

3

24

Reserved

4

32

mh.chksum

Checksum

6

48

mh.data

Message Data
