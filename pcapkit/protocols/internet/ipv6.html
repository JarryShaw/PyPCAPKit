IPv6 - Internet Protocol version 6¶

pcapkit.protocols.internet.ipv6 contains IPv6 only, which implements extractor for Internet Protocol version 6 (IPv6) [*], whose structure is described as below:

Bits

Description

Next Header

Hop Limit

64

Source Address

24

192

Destination Address
