UDP - User Datagram Protocol¶

pcapkit.protocols.transport.udp contains UDP only, which implements extractor for User Datagram Protocol (UDP) [*], whose structure is described as below:

Octets

Bits

Name

Description

0

udp.srcport

Source Port

2

16

udp.dstport

Destination Port

4

32

udp.len

Length (header includes)

6

48

udp.checksum

Checksum
