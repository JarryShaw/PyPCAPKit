Ethernet Protocol¶

pcapkit.protocols.link.ethernet contains Ethernet only, which implements extractor for Ethernet Protocol [*], whose structure is described as below:

Octets

Bits

Name

Description

0

0

eth.dst

Destination MAC Address

1

8

eth.src

Source MAC Address

2

16

eth.type

Protocol (Internet Layer)

+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+                               +
|                         Src MAC Addr                          |
